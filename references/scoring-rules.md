# 冲稳保分差规则与录取概率对照表

> **用途**：advisor Agent 在做志愿排序时使用此规则。三步法的第三步核心——不是凭感觉分档，是按分数/位次差距分档。

---

## 同位分换算规则

### 换算流程

```
当年分数 → 查当年一分一段表 → 省排名 → 用该排名查往年一分一段表 → 往年等效同位分
```

### 硬规则

1. **必须使用省教育考试院官方一分一段表**（T1级数据），禁止使用第三方推算数据
2. **先查位次，再反推分数**——位次是跨年比较的锚点，分数不是
3. **同分人数必须标注**——同一个分数可能对应几百人，位次区间比单一位次更准确
4. **新旧高考切换年份特殊处理**：辽宁 2021 年切换 3+1+2，2021 年之前的数据不可直接对比位次
5. **至少换算近 2 年等效分**（当年 → 去年、当年 → 前年），近 3 年更佳
6. **输出格式**：`当年 X 分 = 省排 Y 名 ≈ 去年 Z 分(省排 W 名) ≈ 前年 V 分(省排 U 名)`

### 示例

```
2025年 490分 = 省排 61,545名 ≈ 2024年 487分(省排 62,100名) ≈ 2023年 484分(省排 62,800名)
```

## ±20分院校匹配规则

### 海选范围

以**等效同位分的中位数**为基准（如三年等效分取中间值），上下浮动 20 分。

| 方向 | 范围 | 目的 |
|------|------|------|
| 向上 +20 分 | 同位分 ~ 同位分+20 | 覆盖"冲"区间的所有可能院校 |
| 向下 -20 分 | 同位分-20 ~ 同位分 | 覆盖"稳"和"保"区间的所有可能院校 |

### 匹配要求

- 每所院校必须标注**近 3 年每年的录取最低位次和分数**
- 如果某年该院校在该省无招生 → 标注"无招生记录"
- 如果某专业是新增专业（无历史数据）→ 标注"🆕 新增专业·无往年数据"
- 专业单独投档分优先于学校最低分

### 输出

候选院校清单（JSON），每所院校包含：
```
{
  school_name, major, level(985/211/双一流/公办/民办),
  scores: {2023: {score, rank}, 2024: {score, rank}, 2025: {score, rank}},
  fluctuation: "稳定(+2)/波动(±8)/大幅波动(±15+)",
  plan_size: N,  // 辽宁招生人数
  data_quality: "T1/T2/T3"
}
```

## 冲稳保四档分类规则

> **核心原则**：第一志愿确保有学上，冲校是锦上添花不是赌命。

### 四档定义

| 档位 | 标识 | 同位分差 | 录取概率 | 说明 |
|------|:---:|----------|:---:|------|
| 🔴 **冲** | 冲刺 | 同位分 **+5 ~ +20** 分 | **20%-40%** | 够得着但不确定，需要运气+大小年波动 |
| 🟡 **稳** | 稳妥 | 同位分 **-5 ~ +5** 分 | **50%-70%** | 正常发挥大概率能上，第一志愿首选区间 |
| 🟢 **保** | 保底 | 同位分 **-5 ~ -30** 分 | **80%-95%** | 发挥失常也能兜住，确保有学上 |
| ⚫ **兜底** | 绝对兜底 | 同位分 **-30 分以下** | **>95%** | 最坏情况的安全网，必须是本省院校 |

### 分类规则

1. **冲校数量 ≤ 总志愿数的 30%**——冲是调味品，不能当主食
2. **第一志愿必须放稳校**（红线四）——开门确保有学上
3. **极小计划(≤4人)不得放前三志愿**——不确定性太高
4. **兜底校必须是省内公办院校**——省外兜底存在计划取消风险
5. **冲稳保之间至少拉开 3-5 分差**——相邻志愿分差太小等于浪费志愿位
6. **稳校区间最多、保底区间次之、冲刺区间最少**——金字塔结构

### 志愿排序模板（7 志愿示例）

```
志愿1：🟡 稳（同位分+2）— 确保有学上
志愿2：🟡 稳/🔴 小冲（同位分+5）— 稳妥中带一点野心
志愿3：🔴 冲（同位分+10）— 需要点运气
志愿4：🔴 冲（同位分+15）— 大小年可能捡漏
志愿5：🟡 稳（同位分-2）— 本地稳校
志愿6：🟢 保（同位分-15）— 发挥小失常兜底
志愿7：⚫ 兜底（同位分-35）— 最坏情况安全网
```

### 录取概率对照表

| 同位分差 | 录取概率 | 档位 | 数据可信度要求 |
|:---:|:---:|:---:|:---:|
| +15 ~ +20 | 20%-30% | 🔴 冲 | T1/T2 必须，无推算 |
| +5 ~ +15 | 30%-40% | 🔴 冲 | T1/T2 必须 |
| 0 ~ +5 | 40%-50% | 🟡 稳（偏冲） | T1/T2 必须 |
| -5 ~ 0 | 50%-70% | 🟡 稳 | T1/T2 必须 |
| -5 ~ -15 | 70%-85% | 🟢 保 | T2 可接受 |
| -15 ~ -30 | 85%-95% | 🟢 保 | T2 可接受 |
| -30 以下 | >95% | ⚫ 兜底 | T2 可接受 |

### 禁止事项

- ❌ 禁止对 ≤4 人计划给出任何百分比概率
- ❌ 禁止基于推算数据做"稳进""空降""百分百"等确定性推荐
- ❌ 禁止让第一志愿为冲校
- ❌ 禁止让兜底校为省外院校或极小计划
- ❌ 冲校之间分差小于 3 分（浪费志愿位）
