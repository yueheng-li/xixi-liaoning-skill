## 维度一：考公考编（公务员/事业单位/教师编）

### 2026年招录概况
- **国考**：招录3.81万人，20,714个职位。税务系统占65.59%（2.5万人）
- **省考联考**：27省招录约16.5万人，较2024年增长8.4%
- **事业编**：全国事业单位年均招录约50万+人
- **教师编**：年均招录约30万+人（人口下降中长期收缩中）
- **硬门槛**：67%国考岗位限应届生，"仅限2026届"占约45%
- **年龄放宽**：国考上限提至38周岁，应届硕博43周岁

### 4年后（2030年）考公趋势

| 趋势 | 影响 |
|------|------|
| **编制总量控制** | 国考扩招速度放缓，财政供养压力加大 |
| **数字化岗位增加** | 计算机/数据类公务员岗位持续增长 |
| **税务系统变化** | AI税务稽查→对基础审核岗冲击最大，但对数据分析/法务岗需求增加 |
| **教师编收缩** | 出生人口下降传导到小学(2025-2028)、中学(2030+)，教师需求进入下降通道 |
| **基层扩招** | 县域/乡镇公务员继续扩招，但城市编制趋于饱和 |

### AI对考公影响
- **低风险**：法官/检察官（AI不可替代审判）、纪检监察（需要人的判断）、综合文秘（AI辅助但不替代）
- **中风险**：税务基础审核（AI自动稽查替代）、行政审批窗口（AI+自助终端替代）
- **应对策略**：选择需要"签字权"、需要"自由裁量"的岗位——AI可以做辅助，但最终责任必须人来承担

## 维度三：外企

### 2026年外企在华校招概况
- **活跃外企**：德企 > 日企 > 美企 > 法企 > 其他
- **热门行业**：汽车/驱动技术 > 医疗器械 > 智能制造 > 食品饮料 > 科技IT
- **招聘特点**：规模不大但培养体系完善，薪资福利好，Work-Life Balance较好
- **薪资参考**：制造业外企起薪8-15K/月，15薪普遍，管培生年薪15W+

### 外企需求专业矩阵

| 梯队 | 专业 | 代表企业 | 特点 |
|------|------|----------|------|
| 🥇 最大需求 | 机械/电气/自动化 | 舍弗勒、西门子、博世、ABB | 德企核心 |
| 🥇 最大需求 | 车辆工程 | 大众、丰田、通用、康明斯 | 新能源转型中 |
| 🥈 最大增量 | 计算机/AI/数据 | 微软、IBM、SAP | 2026年AI岗位井喷 |
| 🥉 稳定需求 | 工业工程/供应链 | 所有制造业外企 | 复合型吃香 |
| 4 | 商科/财务 | 四大(普华永道等)、达能、联合利华 | 管培生不限专业 |
| 4 | 外语（日语N2以上） | 松下、佳能、丰田 | 大连日企集中——日语优势明显 |

### 4年后（2030年）外企趋势

| 趋势 | 影响 |
|------|------|
| **"China for China"深化** | 外企在华不再是销售办，而是研产销一体——需要更多技术人才 |
| **AI岗位继续增长** | 几乎所有外企制造业都在推AI+工业 |
| **地缘政治不确定性** | 部分敏感技术领域外企在华收缩，但消费品/制造业外企不会走 |
| **日韩外企** | 大连的日韩企业集群相对稳定——地理位置+历史纽带 |

### 大连外企红利（重要）

大连是中国日韩外资最集中的城市之一：
- **日语专业+大连** = 独特的地理红利。佳能大连、松下大连每年定向招
- **大连外国语大学日语** = 全国顶尖，毕业生在大连日企有压倒性优势
- **大连理工+大连海事** = 日韩企业研发/工程岗首选
- **风险**：纯语言类正在被AI替代，日语必须+第二技能（经管/技术/IT）

## 维度四：私企/民企

### 2026年私企校招概况
- **"大制造"集群**：机械/设备/重工+工业自动化+汽车——释放超18%新增职位
- **互联网大厂**：HC较2021年收缩约40%，从"招码农"转向"招AI+工程复合型"
- **文科寒冬**：2026年绿牌专业一个文科都没有，制造业私企文科岗从招10人降到0人
- **支持岗过剩**：财务/行政/人力——企业释放8.4%需求，求职端15.7%投递

### AI对私企就业影响（最关键）
- **高风险**：基础码农、基础翻译、基础会计、基础设计、基础文案——已被替代或正在被替代
- **中风险**：数据分析初/中级、HR/行政、初级法律检索
- **低风险**：需要动手操作的（机械维修、电气调试）、需要执业资格的（CPA签字、法律意见书）、需要创造性的（产品架构、算法创新）
- **核心策略**：不学"能被写进prompt"的专业——如果你这行能被"一个指令"描述清楚，AI就能做

## 维度五：个人OPC（独立执业/自由职业/工作室/创业）

### OPC适配度判断标准

| 条件 | 说明 |
|------|------|
| ✅ 有国家执业资格/注册制度 | 医师、律师、CPA、建筑/结构/造价师、执业药师——资格证就是护城河 |
| ✅ 能独立完成闭环 | 一个人+一个助理 = 一个完整业务单元 |
| ✅ 越老越值钱 | 经验积累带来溢价（医生、律师、心理咨询师） |
| ✅ 人身依附性低 | 不依赖特定公司/平台/设备就能产出 |
| ❌ 需要重型资产 | 不适合OPC |
| ❌ 需要大团队协作 | 不适合OPC |
| ❌ 产出可被AI直接替代 | 不适合OPC（会被AI替代，不是OPC的问题） |

| **心理学（临床/咨询方向）** | 心理咨询室 | 3-5年 | 中高(年20-80万) | 极低(情感连接) |
| **建筑学** | 建筑师事务所 | 8-10年 | 高 | 中(AI辅助设计，签字权不可替代) |
| **会计学（CPA）** | 会计师事务所 | 5-8年 | 高 | 中(基础核算被替代，高端审计安全) |
| **药学** | 独立药店/执业药师 | 3-5年 | 中高 | 低 |

#### 🟡 可以OPC（有一定AI替代风险）

| 专业 | OPC路径 | 风险 |
|------|---------|------|
| 计算机/软件 | 独立开发者/外包/产品 | AI替代低端码农——必须做AI做不了的架构/创新 |
| 设计学（非AI方向） | 独立设计师 | AI设计工具冲击——必须做到顶尖才有空间 |
| 翻译 | 自由译者 | ⚠️ AI翻译已商用化，不建议作为主要职业 |
| 新媒体/内容 | 自媒体/KOL | AI内容生成冲击——但有真人IP价值的反而更稀缺 |

#### 🔴 不适合OPC

| 专业 | 原因 |
|------|------|
| 航空航天 | 必须是大型组织 |
| 船舶/海洋工程 | 重型资产+大团队 |
| 化工/冶金 | 重资产+安全监管 |
| 大型机械制造 | 不可能是"一个人"的事 |
| 金融/银行 | 牌照壁垒极高 |

### 4年后（2030年）OPC趋势

| 趋势 | 影响 |
|------|------|
| **AI替代标准化服务** | 基础法律咨询/初级会计/基础翻译将被AI接管 |
| **"真人有温度"的服务溢价** | 心理咨询、中医问诊——越有温度的服务越值钱 |
| **多点执业政策扩大** | 医生多点执业政策宽松→OPC前景更好 |
| **远程服务+AI辅助** | OPC的覆盖半径从3公里扩大到全国（线上问诊/咨询） |
| **平台经济监管趋严** | 纯平台依赖的自由职业（外卖/网约车）利润空间继续压缩 |

### OPC关键建议
> **选OPC路线的黄金组合**：执业资格证 + 5年体制内/大厂积累经验 + 独立执业 = 最优路径
> 不要一毕业就OPC——先在大平台攒够客户/人脉/经验/资本，再独立
> 口腔医学是OPC之王——不受医保控费影响、AI无法替代、不需要夜班急诊、年纪越大患者越多
