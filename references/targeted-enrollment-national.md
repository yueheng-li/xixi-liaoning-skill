# 全品类基层定向赛道汇总

> **统一逻辑**：入学签协议 → 免学费/发补助 → 毕业直接分配 → 带事业编制 → 服务固定年限。
> 和乡村医生完全一致。大众普遍不了解——"入学=入编"是工薪家庭最需要的确定性。
>
> **使用方式**：当考生是工薪/农村家庭+分数在特控线以下+想要确定性编制时，从此文件中按考生适配条件筛选定向赛道。
>
> **更新频率**：每年 6 月（各省定向招生简章发布后）。
> **数据基准**：2025-2026 年各省定向招生公告。

## 通用规则（所有定向统一）

### 录取前提

| 条件 | 说明 |
|------|------|
| **户籍** | 多数要求**本省/本县农村户籍**——城市户口走不通（部分项目对全省开放，需逐省核实） |
| **批次** | 提前批单独志愿——**录取在普通本科批之前，不占用普通批志愿位** |
| **分数** | 分数线普遍比同校非定向专业**低 30-50 分**——定向生是"捡漏"通道 |
| **人数** | 各县名额极度有限（通常 1-5 人/县），但报考人数更少——常出现征集志愿 |

### 在校福利

- **学费全免** + 住宿费减免（省财政/国家承担）
- **每年生活补助**（通常 3000-6000 元/年，视省份和项目）
- 部分项目包实习和考证费用

### 履约要求

- 毕业**必须去指定基层岗位**服务 5-6 年（含培训/规培期）
- 服务期内**不能随意考研**（部分允许在职读研，需单位批准）
- 服务期内**不能跨地区调动**（只能在定向县域内流动）
- 服务期满后可选择：①继续留任（有编制）②考上级单位（有基层工作经验加分）③自由择业

### 违约后果（严重）

- 🔴 **一次性退还全部补助+学费+住宿费**（通常 5-10 万元）
- 🔴 **记入个人诚信档案**——影响后续考公考编
- 🔴 **违约后 3-5 年内不能报考公务员/事业单位**
- ⚠️ 这不是开玩笑的——**签了就是承诺，违约代价极大。** 推荐时务必强调这一点

## 一、医疗基层定向

### 1.1 农村订单定向免费医学生（本科·乡镇卫生院编制）

| 项目 | 详情 |
|------|------|
| **专业** | 临床医学·中医学·中西医临床医学·口腔医学（**五年制本科**） |
| **去向** | 户籍所在县（市/区）**乡镇卫生院**·社区卫生服务中心 |
| **编制** | ✅ **全额事业编** |
| **政策** | 免学费+免住宿费+每年生活补助；毕业后**规培 3 年**（计入服务期）；基层服务 **≥6 年**（含规培） |
| **职能** | 全科诊疗·公共卫生·慢病管理·家庭医生签约 |
| **招聘方式** | 高考填志愿时锁定——提前批单独录取 |
| **适配分数** | 一本线以下至本科线附近（视省份） |
| **开设院校** | 各省中医药大学·医科大学（如辽宁：辽东学院·锦州医大定向等） |

**和乡村医生的区别**：这是**乡镇卫生院编制**（不是村医）。本科+规培+执业医师——完整的医生培养路径，只是工作地点在基层。

**适合**：想学医·接受乡镇工作·分数在本科线至一本线之间·农村户籍优先。

---

### 1.2 乡村医生委托培养专项（专科·村卫生室）

| 项目 | 详情 |
|------|------|
| **专业** | 临床医学（**三年制专科**） |
| **去向** | 本县**各村卫生室**·免试注册乡村医生 |
| **编制** | 村医岗位（非事业编·但享受公卫补贴+基药补助+一般诊疗费） |
| **政策** | 免学费+免住宿费+生活补助（每年约 1 万元）；服务 **5-6 年** |
| **适配分数** | 专科线以上——门槛最低的定向 |
| **招生方式** | 各地卫健委单独招生·高职(专科)提前批 |
| **辽宁项目** | 2025 年辽东学院村医委培 94 人·2026 年预计 187 人 |

**适合**：分数偏低·农村户籍·想要医疗相关稳定工作·能接受长期驻村。

## 二、教育类定向

### 2.1 省属公费师范生（县域中小学·编制全包）

| 项目 | 详情 |
|------|------|
| **覆盖** | 小学全科·初中各科·部分含幼教 |
| **去向** | 户籍所在市/县**公办中小学** |
| **编制** | ✅ **全额事业编** |
| **政策** | **两免一补**（免学费+免住宿费+生活补助）；毕业回户籍市县公办学校；服务 **6 年** |
| **适配分数** | 各省一本线至二本线（省属师范院校） |
| **开设院校** | 各省属师范院校（辽宁：辽师·沈师·鞍师·渤大均有定向名额） |

**核心优势**：毕业回老家当老师——不用考编，直接分配。物理/数学方向的男生在教师招聘中有结构性别红利。

**适合**：想当老师·接受回老家县城/乡镇教书·想要编制。

---

### 2.2 国家优师专项（脱贫县·乡村薄弱学校）

| 项目 | 详情 |
|------|------|
| **覆盖** | 中西部脱贫县·乡村薄弱学校 |
| **培养院校** | **六所教育部直属师范大学**（北师大·华东师大·东北师大·华中师大·陕西师大·西南大学） |
| **编制** | ✅ 全包 |
| **政策** | 免学费+免住宿费+生活补助；服务 **6 年** |
| **户籍** | 限制相对宽松（不一定要求农村户籍） |
| **适配分数** | 985/211 级别（六所部属师范分数线） |

**和公费师范生的区别**：院校层级更高（部属师范 vs 省属师范），去向更偏远（脱贫县 vs 普通县域），分数要求更高。

## 三、农林乡村振兴定向（最冷门·捡漏编制赛道）

### 3.1 公费农科生（乡镇农业服务中心·全额事业编）

| 项目 | 详情 |
|------|------|
| **专业** | 农学·动物科学·农业资源与环境·农林经济管理·农业机械化及其自动化 |
| **去向** | 乡镇**农技推广站**·畜牧兽医站·农产品质量检测站·农村经济管理站 |
| **编制** | ✅ **全额事业编** |
| **政策** | 免学费+免住宿费+生活补助；服务 **5-6 年** |
| **分数优势** | 分数线普遍比同校非定向**低 30-50 分**——常年招不满，常出现征集志愿 |
| **开设省份** | 山东·山西·湖南·四川·吉林·黑龙江等农业大省（各省项目名称不同：公费农科生/基层农技推广定向/乡村振兴专项） |

**为什么是最大的捡漏机会**：家长和考生对"农业"的刻板印象=种地。实际上这些岗位是**全额事业编+技术岗+稳定+轻松**。分数线低 30-50 分=同样分数，别人进普通二本，你进体制内。

**适合**：中等分数·不排斥农业/农村工作·农村户籍优先·想要编制。

## 推荐决策树

```
考生画像 → 扫描七大类定向

① 户籍检查（第一道门槛）：
   ├─ 农村户籍 → 全部定向可选
   ├─ 县城户籍 → 大部分定向可选（医疗/教育/水利/文物/气象）
   └─ 地级市市区户籍 → 仅教育类（部分）·司法类·保密类可选

② 学历层次：
   ├─ 一本线以上 → 订单定向医学生·国家优师·水利/自然资源定向本科
   ├─ 本科线-一本线 → 公费师范生·公费农科生·林业定向·气象定向·文物全科
   ├─ 专科线-本科线 → 乡村医生委培·乡村幼教定向·轨道交通定向
   └─ 专科线以下 → 极少定向可选·需查各省高职定向项目

③ 学科方向：
   ├─ 物理类 → 全部可选（医疗/教育/农林/水利/司法/文物/气象/轨道/保密）
   └─ 历史类 → 教育类·文物全科·基层司法所定向

④ 稳定性需求：
   ├─ 要事业编 → 全部定向（轨道除外）
   ├─ 要行政编（公务员）→ 司法监狱·森林消防·保密
   └─ 要国企稳定 → 轨道交通

⑤ 环境偏好：
   ├─ 能接受乡镇/偏远 → 全部可选
   ├─ 必须县城 → 气象·文物·司法监狱·教育类（县城学校）
   └─ 必须地级市以上 → 仅轨道交通（地铁公司在地级市）

⑥ 家庭接受度：
   ├─ 家人支持去基层 → 全部可选
   └─ 家人强烈反对 → 仅推荐轨道交通·保密（非基层岗）

⑦ 违约风险评估（必须告知）：
   「你签了协议，毕业就必须去服务6年。中途毁约=退全部补助+诚信档案黑名单+3-5年不能考编。你确定能接受吗？」
```

## 推荐通用规则

1. **定向推荐优先级最高**：对于工薪/农村家庭+分数在本科线至一本线之间的考生，定向赛道优先于所有其他推荐——因为"入学=入编"的确定性是任何其他路径无法提供的。

2. **必须逐条告知违约后果**：这不是"建议"，是法律约束的协议。推荐时必须完整告知：退还全部费用+诚信档案+禁考期。让考生和家长在充分知情的前提下做决定。

3. **农村户籍是第一道筛选**：先确认户籍类型，再推荐对应定向。城市户口推荐不了农村定向——不要浪费时间。

4. **提前批不占普通批志愿位**：定向多在提前批录取。本科普通批志愿照填——定向录了就定向，定向没录不影响普通批。**多一条命。**

5. **分数线低 30-50 分是普遍规律**：定向因为要服务基层，报的人少——这是"捡漏"的本质。不要因为分数低就怀疑它的编制真实性。

6. **"常年招不满""征集志愿"是推荐关键词**：这些词最能打动犹豫的家长——"别人不知道=你的机会。"
