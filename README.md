## 这是什么

一个专注于**辽宁省**的高考志愿推荐 Claude Code Skill。以张雪峰的认知框架为分析基底，采用**核心三步法**（①等效同位分换算 → ②±20分院校海选 → ③冲稳保志愿排序），整合辽宁省 64 所本科院校数据库、22 个重点产业集群人才需求、4 年就业前瞻（含 AI 冲击分析），从**考公考编/国企央企/外企/私企/个人OPC/医学医疗**六个维度逐项拆解，生成带配图建议的 11 章完整 Markdown 报告。

## 核心特性

### 📋 核心三步法
1. **等效同位分换算**：一分一段表 → 分数转位次 → 往年等效分反推
2. **±20分院校海选**：同位分±20分范围内，搜索3年有录取记录的所有本科院校
3. **冲稳保志愿排序**：四档分差（冲+5~+20/稳±5/保-5~-30/兜底-30↓）+ 录取概率标注

### 🏗️ Pipeline 多Agent 架构
- **data-fetcher Agent**：等效分换算 + ±20分海选 + 考公排行榜 + T1-T4数据分级 + 双源交叉验证
- **advisor Agent**：六维度就业分析 + 冲稳保四档分类 + 张雪峰视角 + 质量红线自检
- **report-writer Agent**：三步法速览卡片 + 11章固定模板 + 三档表达（🔴全力/🟡温和/🟢共情）+ 最后提醒强制声明
- **quality-auditor Agent**：独立审查（8项一票否决 + 6项扣分 + 6项v3.2新增检查）

### 📊 辽宁深度绑定
- 42 所公办 + 22 所民办本科院校完整数据库
- 22 个重点产业集群人才需求
- 大连/沈阳/辽宁其他城市/全国四层优先级推荐
- 跨省推荐强制披露回辽宁就业短板

### 🧊 小众冷门赛道
- 七大类 26 条全国小众赛道（认知偏见 = 就业护城河）
- 七大类 18 条定向委培赛道（入学 = 入编）
- 辽宁四大定向体系详解（村医 / 公费师范 / 水利 / 农科）

### 🛡️ 质量保障
- 4 条质量红线 + 8 项一票否决 + 6 项扣分评分
- 所有录取数据标注 T1-T4 可信度等级
- 分数线双源交叉验证硬规则
- 独立质量审计 + 审计报告存档

## 触发方式

在 Claude Code 中输入以下任意关键词即可激活：

- 「辽宁高考志愿」
- 「辽宁省大学推荐」
- 「辽宁选专业」
- 「我想在辽宁上学」
- 「辽宁考生」
- 「雪峰视角看辽宁」
- 「辽宁什么专业好」

## 标准输入格式

| 字段 | 必填 | 说明 |
|------|:---:|------|
| 学生姓名 | ✅ | — |
| 性别 | ✅ | 男 / 女 |
| 户口省份 | ✅ | — |
| 考试科目 | ✅ | 物理类 / 历史类 |
| 成绩（分数） | ✅ | 高考总分 |
| 期望专业 | 选填 | 不限则全专业推荐 |
| 考虑地点 | 选填 | 默认大连→沈阳→辽宁其他→全国 |
| 家庭条件 | 选填 | 默认四档分别建议 |
| 考研打算 | 选填 | 默认两条路径 |
| 行业偏好 | 选填 | 默认技术型优先 |
| 关注维度 | 选填 | 默认全部6个维度 |

## 案例演示

### 吴磊 · 490分 · 辽宁物理类 · 工薪家庭 · 倾向军警/师范

> 完整报告见仓库 `examples/` 目录或点击查看：[吴磊_高考志愿推荐报告_物理类_490分_v3](https://github.com/yueheng-li/xixi-liaoning-skill/blob/main/examples/吴磊_高考志愿推荐报告_物理类_490分_2026-06-18_v3.md)

**考生画像**：男 · 物理类（物政）· 490分 / 省排 6.1万 · 超本科线 123 分 · 工薪家庭 · 偏好军警/师范/政法。

**关键纠错**：过程中发现辽宁警察学院有两条录取线——提前批公安类（536分·可联考入警）和本科批普通类（483分·不可联考）。吴磊 490 分只能上本科批普通类，**不能参加公安联考**。这个信息差纠正后，推荐策略从"警校优先"完全翻转为"师范+定向军士+参军"。

**最终推荐路径**：

| 优先级 | 路径 | 确定性 | 说明 |
|:---:|------|:---:|------|
| 🥇 | 定向培养军士 | 极稳 | 专科提前批·490 分远超所有军士院校线·体检政审过即稳·不占本科志愿位 |
| 🥈 | 大学期间参军 | 极高 | 大二走→退役→考研加分/考公定向/免试专升本→两年换终身退役士兵标签 |
| 🥉 | 师范入编（社招路线） | 中等 | 物理教师紧缺+男教师结构性别红利·但需认真准备社招笔试+试讲 |
| 🏅 | 航海技术（渤海大学提前批） | 极稳 | 455分/7.9万名·位次之上1.7万名·跑船5年攒80万·提前批不占普通志愿 |
| 🏅 | 特殊教育（绥化学院） | 稳偏冲 | 493分/6万名·编制+15%特教津贴·全国缺口10万·竞争比普教低5倍 |
| 🏅 | 现代殡葬技术（长沙民政） | 极稳 | 480分·专科带编率最高·永不失业·需家人理解 |

**报告包含**：11 章完整分析 · 考生定位 · 张雪峰视角 · 考公排行榜 · 7 个推荐学校/专业 · 六维度对比 · 志愿排序（7 位并行赛道）· 5 条就业路径 · 避坑清单 · 家庭四档建议 · T1-T4 数据等级标注。

**质量审计**：8 项一票否决全部通过 · 扣分项 9.5/10 · 独立审计报告存档。
