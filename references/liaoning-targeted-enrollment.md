# 辽宁省定向赛道精简清单

> **统一逻辑**：入学签协议 → 免学费/发补助 → 毕业直接分配 → 带事业编制（村医除外）→ 服务固定年限。
> 辽宁四大定向体系：村医/定向医、公费师范、公费农科、基层水利。
>
> **使用方式**：当考生为辽宁户籍+工薪/农村家庭+分数在特控线以下时，按户籍类型和职业偏好从此文件筛选推荐。
>
> **更新频率**：每年 6 月（辽宁省定向招生简章发布后）。
> **数据基准**：2026 年辽宁省定向招生公告。

## 报考通用规则

| 条件 | 说明 |
|------|------|
| **户籍** | 绝大部分定向要求 **辽宁县/县级市农村户籍**（新民市·瓦房店市·海城市·东港市·开原市·北票市等）。**水利定向无农村户籍限制**——城市户口也可报。 |
| **批次** | 村医/定向医→**专科提前批**；公费师范/公费农科→**本科提前批**；水利定向→**普通本科批平行志愿** |
| **在校福利** | 免学费+免住宿费+每年生活补助（3000-6000 元/年，视项目） |
| **违约后果** | 🔴 退还全部补助+学费+住宿费 · 🔴 记入诚信档案 · 🔴 服务期内不能考公考编考研（脱产） · 🔴 不能跨地区调动 |

## 一、医疗基层定向

### 1.1 乡村医生委托定向（村卫生室·无编制）

| 项目 | 详情 |
|------|------|
| **院校** | **辽东学院**（丹东·公办） |
| **专业** | 临床医学（**专科三年制**） |
| **2026 计划** | **65 人** |
| **批次** | 高职（专科）**提前批** |
| **报考条件** | 辽宁**县域农村户籍**·过专科线即可 |
| **福利** | 免学费+免住宿费+每年 **1 万元**生活补助 |
| **就业** | **村卫生室**·县管乡聘村用·发乡村医生证 |
| **岗位补贴** | 省财政每年 **1.2 万元**岗位补贴（额外·在工资之外） |
| **服务年限** | 最低 **6 年** |
| **编制** | ⚠️ **无事业编**——合同管理 |

**和定向医学生的核心区别**：这个是去**村卫生室**（村里），不是去乡镇卫生院。有岗位补贴但无编制。

**适合**：辽宁县域农村户籍·分数偏低（专科线以上）·想回老家村里当村医·能接受驻村。

**报名条件**：户籍为辽宁省县（含县级市）范围内。高考成绩达专科线以上。将辽东学院填在专科提前批第一顺序志愿。

**推荐话术**：「专科提前批——录取在本科批之后。你本科志愿照填，本科录了就上本科；本科全滑了，村医委培兜底。免学费+每年 1 万补助+每年 1.2 万岗位补贴——读三年倒赚钱。」

### 1.2 农村订单定向免费医学生（乡镇卫生院·有编制）

| 项目 | 详情 |
|------|------|
| **院校** | **辽东学院**（丹东·公办） |
| **专业** | 临床医学（**专科三年制**） |
| **2026 计划** | **187 人** |
| **批次** | 高职（专科）批**第一顺序志愿** |
| **报考条件** | 辽宁**县域农村户籍**·过专科线 |
| **福利** | 免学费+免住宿费+生活补助 |
| **就业** | 户籍所在县**乡镇卫生院**·全科岗位 |
| **编制** | ✅ **全额事业编** |
| **履约** | 毕业后**规培 2 年**（计入服务期）·基层服务 **6 年** |

**和乡村医生委培的核心区别**：这个是去**乡镇卫生院**（有编制）——不是去村里。有规培+事业编+完整的全科医生发展路径。

**适合**：辽宁县域农村户籍·分数偏低·想进医疗系统·要事业编·接受乡镇工作。

**推荐话术**：「专科三年+规培两年=五年后你是乡镇卫生院事业编全科医生。不用卷临床医学五年制+考研+三甲规培那条路。确定性极高——代价是乡镇六年。」

## 二、辽宁省公费师范生

### 2.1 省属公费师范生（县域中小学·事业编）

| 项目 | 详情 |
|------|------|
| **培养院校** | **沈阳师范大学·辽宁师范大学·渤海大学·鞍山师范学院·沈阳大学等** |
| **专业** | 小学教育·汉语言文学·数学与应用数学·英语·学前教育·物理学·化学等 |
| **批次** | 本科**提前批** |
| **政策** | **两免一补**（免学费+免住宿费+生活补助） |
| **就业** | 毕业回生源地**县城/乡镇中小学**·公办教师 |
| **编制** | ✅ **全额事业编** |
| **服务年限** | **6 年** |

**适合**：辽宁户籍·想当老师·接受回老家县城/乡镇教书·要编制。

**推荐话术**：「别人考教师编挤破头——你高考填志愿时就锁定了。毕业直接回老家当老师，事业编，不用考。代价是回县城服务六年——但这六年在哪不是教书？」

## 三、基层水利人才定向（2026 新开·沈阳农大本科）

| 项目 | 详情 |
|------|------|
| **院校** | **沈阳农业大学**水利学院（沈阳·公办·省属重点） |
| **层次** | **四年制本科** |
| **专业** | 水利水电工程·农业水利工程 |
| **2026 计划** | **80 人**（四年总计定向培养 320 人） |
| **批次** | **普通本科批平行志愿** |
| **户籍** | 🟢 **无农村户籍限制**——城市户口也可报 |
| **福利** | 学费+住宿费+教材费**财政补助**·正常申领奖助学金 |
| **就业** | 定向县域**水利局**·**乡镇水利服务中心** |
| **编制** | ✅ **全额事业编** |
| **服务年限** | **6 年**·服务期不可脱产读研、不可跨县调动 |

**为什么是 2026 年最大的定向捡漏机会**：
- 2026 年**新开项目**——第一年招生，知晓度为零
- **不限农村户籍**——城市户口也能报（其他定向几乎都限农村户籍）
- **普通本科批**——不是提前批，不跟其他定向抢志愿位
- **沈阳农大**——省属重点，比专科定向层次高
- 辽宁农村水利设施老化严重——基层水利人才断档二十年

**适合**：辽宁户籍（不限城乡）·物理类·分数在本科线至一本线之间·想要事业编·理科·能接受县城/乡镇水利局工作。

**推荐话术**：「2026 年新开的定向——第一年招生，根本没人知道。不限农村户籍——这是所有定向里独一无二的。沈阳农大本科+全额事业编+水利局。辽宁农村水利设施老化二十年没人修——你就是去修的人，也是去拿编制的人。」

## 四、公费农科生（基层农技站事业编）

| 项目 | 详情 |
|------|------|
| **辽宁模式** | 基层农技 **"三定向"**（定向招生+定向培养+定向就业） |
| **主力院校** | **沈阳农业大学**（本科）·**辽宁农业职业技术学院**（专科）·**辽宁生态工程职业学院**（专科） |
| **专业** | 农学·园艺·动物医学·农业机械化及其自动化·农林经济管理 |
| **批次** | 本科/专科**提前批** |
| **政策** | 免学费+生活补助 |
| **就业** | 生源县**乡镇农业技术推广站**·**畜牧兽医站** |
| **编制** | ✅ **全额事业编** |
| **服务年限** | **5 年** |
| **特点** | **常年招不满**·分数线普遍低于普通专业 30-50 分·农村户籍优先 |

**为什么是最大的捡漏机会**：
- 家长和考生对"农业"的刻板印象=种地。实际上这些岗位=全额事业编+技术岗+稳定+轻松
- 常年招不满→分数线低于同校普通专业 30-50 分
- 同样分数，别人进普通二本，你进体制内

**适合**：辽宁县域农村户籍·中等分数·不排斥农业技术工作·要编制。

**推荐话术**：「你以为学农业是种地？乡镇农技站是全职事业编——坐办公室的。常年招不满，分数线比普通专业低 30-50 分。同样的分数，别人去二本，你进体制内。这就叫信息差。」
